GitUM - Git User Manager

GitUM is a git user experience tool. It can be useful to manage your local git users. You can add users to the user manager and easily switch between them with one simple command.

GitUM currently supports Ubuntu, and is under development in order to add useful functionalities and support to diverse systems.

You can easily install GitUM with these two commands in your console: sudo wget -O /usr/local/bin/gitum https://github.com/mmidu/gitum/raw/master/build/gitum sudo chmod +x /usr/local/bin/gitum
