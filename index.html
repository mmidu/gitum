GitUM

Git User Manager

GitUM is a user management tool for Git.

It can be used to manage your local git users.

Thanks to GitUM You can add users to the user manager and easily switch between them with simple commands.

GitUM is currently under development. As for now, it's tested on Ubuntu.

We hope we can get to add useful functionalities and support to diverse systems real soon!

You can easily install GitUM with these two commands in your console:

sudo wget -O /usr/local/bin/gitum https://github.com/mmidu/gitum/raw/master/build/gitum sudo chmod +x /usr/local/bin/gitum

You can even make it a one-liner 😜

sudo wget -O /usr/local/bin/gitum https://github.com/mmidu/gitum/raw/master/build/gitum && sudo chmod +x /usr/local/bin/gitum

GitUM is really easy to use.

First of all, you can initialize it. This will also import the currently active local Git user. gitum init ðŸŽ‰ It's as simple as that!

After that, you can add a user gitum add you just need to follow the prompt.

You can also delete a user, through the identifier gitum remove --identifier identifier check the current user gitum current list the available users gitum list and set one of them as active gitum set username here you need to replace username with the user identifier you want to set active.

If you enable a different git user, not through GitUM, you can import it in GitUM gitum import And lastly, if you need any, you can ask for help gitum help there, you can also learn the short version of these commands.
